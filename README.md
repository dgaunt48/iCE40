# iCE40
In Progress iCE40 FPGA Projects.


## KiCad\Ice40hx4k_tq144
A simple custom iCE40 FPGA board with lots of level shifters I use when prototyping.


## VIA_6522
6522 VIA iCE40 FPGA Core is taking shape.  A real Vic20 will now boot to BASIC with the FPGA running the timer for the cursor blink and performing the keyboard matrix scan.

## VIC_6560
Initial test to use the VIC hardware register values and memory dumps to create a VGA display.
